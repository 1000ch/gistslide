# GistSlide

## About

Inspired by https://github.com/nzoschke/gistdeck.

## Usage

+ Visit [here](http://gistslide.herokuapp.com/) and add link as bookmarklet.
+ Go to gist which you want to present.
+ Click bookmarklet.
