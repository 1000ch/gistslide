GistSlide

GistSlide is simple presentation tool for gist.
Inspired by GistDeck.
You can use this with only 3 steps !!!

Usage

  1. Drag following label to bookmarkbar.
  2. Go to gist which you want to present.
  3. Click bookmarklet.
